smart_toy Chrome AI Sidebar

A beautiful, performant Chrome extension that integrates Google's built-in AI directly into your browser with full page context awareness.

Features

  • speed Local & Fast - On-device processing with Gemini Nano
  • privacy_tip Private - No data sent to external servers
  • article Context-Aware - Include page content in queries
  • palette Material Design 3 - Beautiful, modern UI
  • stream Streaming - Real-time response generation
  • summarize Smart Summaries - Built-in summarization API
Material Design 3

On-device AI offers several key benefits:

2. Speed - no network latency
3. Offline - works without internet
4. Cost - no API fees
